# HR-Analytics-Power-Bi-Dashboard
An interactive HR Analytics Dashboard built using Power BI to analyze employee workforce data, attendance, salary distribution, and departmental insights.

# Project Overview
This dashboard provides a comprehensive view of:
- Total Employees
- Average Salary
- Attendance (Days Present / Absent)
- Leave Analysis
- Department-wise Employee Distribution
- Gender Distribution
- Salary by Experience Level
- Headcount by Country
- Salary Trends by Age

# Tools Used
Power BI
- DAX (Data Analysis Expressions)
- Data Modeling
- Excel / CSV Dataset
- Data Cleaning & Transformation

# Dashboard Features
- Interactive Filters (Year, Gender, Department, Country)
- Salary Trend Analysis
- Employee Growth Tracking
- Attendance Analysis by Year & Gender
- Department-wise Salary Comparison
- Country-wise Headcount Visualization
- Detailed Employee Information Table
